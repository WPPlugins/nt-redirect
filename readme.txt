Make redirect page and automatically insert 'rel=nofollow' and 'target=_blank' to all the external links.

== Description ==

Just simple, if you use this plugins, all the external links will be automatically add 'rel=nofollow', 'target=_blank' and make redirect page.

= Feature =
* Add 'rel=nofollow' for all the external link.
* Add 'target=_blank' for all the external link.
* If you already added 'rel=dofollow' in links, this plugins will not add 'rel=nofollow'.

You can visit our website <a href="http://hutruc.com/wordpress/wordpress-plugins/nt-redirect-tao-trang-chuyen-huong-cho-lien-ket-ngoai.html">hutruc.com</a>

Wating for your valuable feedback/comments.

== Installation ==

1. Download the latest zip file and extract the 'nt-redirect' directory.
2. Upload it to your '/wp-content/plugins/' directory.
3. Activate 'NT Redirect' on the 'Plugins' menu in WordPress.
4. That's it, you are all done :)
